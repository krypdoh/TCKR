TCKR

Stock & Crypto Ticker for Windows

About

TCKR is a customizable, always-on-top stock and cryptocurrency ticker for Windows. It displays real-time prices, supports multiple monitors, and offers a modern, glassy UI with LED effects.

Features

  • Real-time stock and crypto prices (Finnhub & CoinGecko APIs)
  • Customizable appearance and scroll speed
  • Multi-monitor support
  • System tray integration
  • Easy stock/crypto management
  • Command-line options

Installation

  1. Clone or download this repository.
  2. Install Python 3.8+ and required dependencies: 
    pip install -r requirements.txt
  3. Run TCKR.py: 
    python TCKR/TCKR.py

On first launch, you will be prompted for your Finnhub API key.

Configuration

Use the right-click menu or system tray icon to access settings, manage tickers, and move the ticker between screens.
