# TCKR-v1.1.0 Build & Run Checklist

## Required Python Source Files
- TCKR-v1.1.0.py  *(main application)*
- modern_gui_styles.py
- memory_pool.py
- ticker_utils_numba.py
- ticker_utils_cython.pyx *(and compiled .pyd/.so, see below)*
- ticker_utils_cython.c *(generated by Cython, not edited directly)*

## Build & Dependency Files
- requirements.txt  *(Python dependencies)*
- setup_cython.py   *(for building ticker_utils_cython)*

## Resources & Assets
- TCKR.ico
- notify.wav
- neon_check.png
- neon_cross.png

## Optional / Packaging Files
- TCKR.spec
- TCKR-console.spec  *(for PyInstaller builds)*

## Build Notes
- The app creates and uses user-specific settings and stock files in `%APPDATA%\TCKR` (not in this folder).
- `ticker_utils_cython.pyx` must be built with `setup_cython.py` (requires Cython, numpy, and a C compiler).
- If `ticker_utils_cython` is not built, `ticker_utils_numba.py` is used as a fallback.

## Not Required
- Old builds, docs, Output, build, dist_obf, .csv, .md, .txt (except requirements.txt), or __pycache__ are NOT needed to run the app.
